## Documentation

You can see below the API reference of this module.

### `gmInstalled(fn)`
Check if GraphicsMagick is installed.

#### Params

- **** `fn`: An optional callback function.

#### Return
- **Boolean** `true` if the `gm` binary is detected, `false` otherwise.
